Query API¶

POST /sql¶

Search database

Form Parameters:
 
  • query – JSON encoded query
Request JSON Object:
 
  • method (string) – one of sql, chemsql
  • targets (object) – list of resource entity
  • key (string) – query column key
  • values (object) – list of query value
  • operator (string) – eq, gt, lt, ge, le, in, lk
Status Codes:
  • 200 OK – no error

Request JSON Object:
 
  • method (string) – one of prop, exact, sub
  • flush (boolean) – if true, use map function with single process
  • targets (object) –
  • key (string) –
  • values (object) –
  • operator (string) –
  • format (string) – one of smiles, molfile, dbid
  • value (string) – query value
  • querySource (string) –
  • threshold (float) –

Status Codes:
  • 200 OK – no error
POST /graph¶

Submit graph connection calculation job

Form Parameters:
 
  • query – JSON encoded query
Request JSON Object:
 
  • nodeTableId (string) – smiles | molfile | dbid
  • indices (object) – list of index
  • molecules (object) – list of molecule object
  • measure (string) – gls | morgan
  • threshold (float) – graph connection score threshold
  • ignoreHs (boolean) – if true, ignore explicit hydrogens
  • diameter (int) – MCS-DR parameter
  • maxTreeSize (int) – MCS-DR parameter
  • molSizeCutoff (int) – MCS-DR parameter
Status Codes:
  • 200 OK – no error
POST /sdf¶

Structure image preview

Form Parameters:
 
  • contents – request file object
  • columns – list of SDF option attribute to show
  • implh – if 1, make hydrogens implicit
  • recalc – if 1, recalculate 2D coordinates
Status Codes:
  • 200 OK – no error

Form Parameters:

Request JSON Object:
 
  • id (string) – datatable id

Status Codes:
  • 200 OK – no error

Response JSON format¶

Datatable response:

{
    "id": "hoge",
    "format": "datatable",
    "progress": 100
}
Attributes:
  • id(string) - datatable ID
  • format(string) - “datatable”
  • startDate(string) - date computation job started
  • responseDate(string) - date of response sent from server
  • execTime(float) - execution time
  • searchCount(int) - sum of the number of rows to search
  • searchDoneCount(int) - sum of the number of rows searched
  • progress(float) - 100
  • recordCount(int) - number of rows found
  • dataSource(string) - list of database entity
  • query(object) - search query
  • status(string) - one of “Queued”, “Completed”, “In progress”, “Failure”, “Aborting”, “Aborted”
  • records(object) - list of record
  • columns(object) - list of columns
  • extraColumns(object) - list of extra columns
