Web API¶

POST /sql¶

Search database

Form Parameters:
 
  • query – JSON encoded query
Request JSON Object:
 
  • method (string) – one of sql, chemsql
  • targets (object) – list of resource entity
  • key (string) – query column key
  • values (object) – list of query value
  • operator (string) – eq, gt, lt, ge, le, in, lk
Status Codes:
  • 200 OK – no error

POST /graph¶

Submit graph connection calculation job

Form Parameters:
 
  • query – JSON encoded query
Request JSON Object:
 
  • nodeTableId (string) – smiles | molfile | dbid
  • indices (object) – list of index
  • molecules (object) – list of molecule object
  • measure (string) – gls | morgan
  • threshold (float) – graph connection score threshold
  • ignoreH (boolean) – if true, ignore explicit hydrogens
Status Codes:
  • 200 OK – no error
POST /sdf¶

Structure image preview

Form Parameters:
 
  • contents – request file object
  • columns – list of SDF option attribute to show
  • implh – if 1, make hydrogens implicit
  • recalc – if 1, recalculate 2D coordinates
Status Codes:
  • 200 OK – no error
POST /rows¶

Submit graph connection calculation job

Form Parameters:
 
  • query – JSON encoded query
Request JSON Object:
 
  • id (string) – datatable id
  • command (string) – fetch | abort
Status Codes:
  • 200 OK – no error
